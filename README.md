#  Web Insight Scraper

A powerful Python tool to extract, analyze, and export website structure data (headings & links) into CSV files.

---

# Features
- Extract H1, H2, H3 headings
- Clean and structured data output
- Export results to CSV
- Handles full URL paths
- Fast and lightweight

---

# 🛠️ Tech Stack
- Python
- BeautifulSoup
- Requests
- Pandas

---
